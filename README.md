Instagram Media Extractor is a Firefox extension that lets you download the currently visible media from Instagram posts, reels, carousels, and stories.

## Features
- Download full-resolution photos from posts, carousels, and stories.
- Download videos from reels, video posts, and stories.
- Uses the active carousel item, so it saves exactly what is currently visible.
- One-click popup action: **Download Current Media**.

## How to use
1. Open Instagram and navigate to a post, reel, story, or carousel item.
2. If it is a carousel, switch to the exact item you want.
3. Click the extension icon in Firefox.
4. Click **Download Current Media**.

## Compatibility
- Firefox Desktop
- Firefox for Android (manifest includes Android compatibility metadata)

## Privacy
The extension does not collect or transmit user data.
